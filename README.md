# SIMPLE_MIPS_CPU
A simple MIPS CPU, 32bits, designed with verilogHDL.  

**ALU----100%.**  
**REGFILE----100%.**

Files:  
-----
**ALU（ALU\ALU.srcs）:**  
1. ALU.v: ALU  
2. LOGIC.v: AND/OR/XOR/LUI    
3. MATH.v: ADD/SUB  
4. SHIFT.v: SLL,SRL,SRA  
5. ALU_TB.sv: TestBench for ALU.  

**REGFILE（REGFILE\REGFILE.srcs）:**  
1. REGFILE.v: REGFILE  
5. REGFILE_TB.sv: TestBench for REGFILE.
